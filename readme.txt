The file-uploader projects serves as a stepping stone towards Django. The goal is to create a web-application which allows the (logged in) user to upload any item to a database. Django keeps track of each file and it's affiliated user/organization

Useful links:
https://mkdev.me/posts/fundamentals-of-front-end-django (tutorial)
https://startbootstrap.com/template/sb-admin (front-end template)
